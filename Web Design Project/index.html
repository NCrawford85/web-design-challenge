Lattitude Vs. X

This project analyzed weather data from over 500 cities to analyze how weather patterns shift depending on location lattitude. We used the OpenWeatherMap API to generate the data set and then analyzed the results based on temparture, humidity, windiness, and cloudiness compared against lattitude.
